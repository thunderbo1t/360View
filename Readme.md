# Pre Requisites
  * Ubuntu 16.04
  * OpenCV 3.X
  * Python 2.X
  * flask latest

# Installation
  * Install python, pip and flask in the system 
    ```bash
    sudo apt install python python-pip  python-opencv
    sudo pip install flask
    ```
  
# Usage
  * Go to the source code directory
  `cd postserver`
  * Run the command `python app.py`
